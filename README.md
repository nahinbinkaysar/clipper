# Clipper

A native Windows clipboard utility written in **C++**.

Clipper is a lightweight on-screen clipboard utility designed for pen, touch, and tablet-based workflows.

It allows quick access to copy and paste actions without relying on a physical keyboard.


## Why Clipper?
When using a device in tablet or 360Â° mode, switching between pen, mouse, and keyboard can break your workflow.  
Clipper solves this by placing essential clipboard controls directly on the screen.

- Native Windows app (no frameworks)
- Written in **C++**
- Built with **Visual Studio**
- Executable size: **815 KB**
- Memory usage: **~5.1 MB RAM**

## Features
- Floating on-screen copy & paste buttons
- Duplicate button to make copy-pasting easier
- Optimized for pen and touch input
- Works system-wide
- Minimal, distraction-free UI
- Lightweight and fast
